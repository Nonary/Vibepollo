# Vibepollo 1.15.4 - 2026-05-03

Notice: Vibepollo may trigger false-positive antivirus alerts. Details and workarounds are documented here: https://github.com/Nonary/vibepollo/issues/29

## Changes
- Improved Windows Graphics Capture delivery with event-driven IPC to reduce polling and keep captured frame metadata aligned with shared textures.
- Improved source builds by caching libwebrtc in a shared per-user dependency directory and documenting the faster quickstart path.
- CI/CD and Releases are now automated, and all components such as SudoVDA are now included in the build to make contribution easier.

## Fixes
- Fixed standard-mode Playnite streams ending when Playnite closes after launching a game.
- Fixed virtual display cleanup and physical display restore handling so app exit, reconnect, and paused-session transitions do not leave stale virtual displays or overwrite pending monitor restores.
- Fixed Steam Streaming Speakers restore behavior when the original audio endpoint is temporarily missing between streams.
- Fixed Windows autostart crashes by deferring the startup encoder probe until the interactive desktop is ready.
- Fixed display helper watchdog exceptions from terminating Vibepollo during pause, teardown, wake, or sleep transitions.
- Fixed AV1 HDR support not being advertised to clients when an encoder supports AV1 Main10 HDR.
