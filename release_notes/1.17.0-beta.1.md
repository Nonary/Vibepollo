# Vibeshine 1.17.0-beta.1 - 2026-06-13

Notice: Vibeshine may trigger false-positive antivirus alerts. Details and workarounds are documented here: https://github.com/Nonary/vibeshine/issues/59

## Changes
- Added opt-in NVIDIA RTX HDR streaming support so compatible NVIDIA hosts can synthesize HDR output from SDR games for HDR-capable clients. The feature uses per-app RTX HDR settings, foreground app matching, and safe fallbacks when the required NVIDIA runtime, GPU support, HDR stream mode, or shim components are unavailable.
- Updated the experimental Vibeshine Display Driver package to libvirtualdisplay 1.4.0, adding Vulkan HDR implicit layer support for virtual displays.
