# Vibepollo 1.17.0-beta.6 - 2026-06-15

## Changes
- Added real-time RTX HDR adjustment to help calibrate HDR while streaming; when NVIDIA Overlay does not appear during a stream, settings can be fine-tuned from another device such as a mobile phone.

## Fixes
- Kept RTX HDR streams stable when switching between the desktop and opted-in apps, avoiding unnecessary capture or encoder mode changes.
- Capped RTX HDR peak-brightness controls at the conversion ceiling so UI settings and streaming metadata stay consistent.
- Stabilized RTX HDR stream metadata across encoder paths so real-time tuning does not cause client tone mapping to drift while streaming.
- Updated the virtual display driver integration with DPI retention fixes and recent HDR/display reliability improvements.
- Rejected unsigned virtual display driver catalogs during install validation so bad refreshed driver artifacts fail before installation.
