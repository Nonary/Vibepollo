# Vibepollo 1.15.5-alpha.3 - 2026-05-08

Notice: Vibepollo may trigger false-positive antivirus alerts. Details and workarounds are documented here: https://github.com/Nonary/Vibepollo/issues/59

## Changes
- Testing SignPath code signing for the Windows installer artifacts. This should help reduce false-positive Microsoft Defender and SmartScreen warnings while the signing reputation builds over time.
- Improved installer recovery for PCs where registry cleaning or uninstall-cleanup software damaged the Windows Installer cache. These tools often do more harm than good and can corrupt installer state that Windows needs for upgrades, repairs, and uninstalls.

## Fixes
- Fixed upgrade and downgrade button labels when installing minor revisions of the same Vibepollo version, so newer payloads show Upgrade Vibepollo and older payloads show Downgrade Vibepollo instead of falling back to Reinstall Vibepollo.
