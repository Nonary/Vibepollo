# Vibepollo 1.17.0-beta.8 - 2026-06-20

Notice: Vibepollo may trigger false-positive antivirus alerts.

## Changes
- Updated Windows release packaging to require the pinned TrueHDR runtime in release and tester installers, so RTX HDR support ships with the DLLs it needs.
- Refreshed the packaged virtual display driver to libvirtualdisplay 1.4.3.
- Improved the Windows signing and publishing flow so MSI, bootstrapper, symbol, and release assets can be finalized more reliably.

## Fixes
- Started RTX HDR conversion from the app override immediately for matched games instead of waiting on NVIDIA App profile activation state.
- Created encoder-probe temporary virtual displays on the configured render adapter, preserving capture setup on dual-GPU hosts.
