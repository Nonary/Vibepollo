# Vibepollo 1.16.0-beta.1 - 2026-05-28

Notice: Vibepollo may trigger false-positive antivirus alerts. Details and workarounds are documented here: https://github.com/Nonary/Vibepollo/issues/59

## Changes
- Synced Vibepollo with the current Sunshine upstream so the project can keep pace while Apollo has not received updates recently; going forward Vibepollo will inherit Sunshine changes in addition to Vibeshine changes.
- Added a Dashboard warning when the saved display snapshot is stale and should be recreated.

## Fixes
- Fixed HEVC Main10 and AV1 10-bit handling with NVENC API 12.1 drivers by preserving the legacy bit-depth layout expected by older runtimes.
- Fixed HDR and SDR setup so AV1 10-bit fallback, launch HDR intent, and delayed display-helper re-applies keep the intended color mode.
- Fixed stream-end display restoration to prefer the golden snapshot before previous session state, avoiding stale restores that could leave secondary monitors disabled.
- Fixed display capture overrides so physical output selections are preserved when display configuration is disabled.
