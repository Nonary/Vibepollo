# Vibeshine 1.17.0-beta.7 - 2026-06-19

Notice: Vibeshine may trigger false-positive antivirus alerts. Details and workarounds are documented here: https://github.com/Nonary/vibeshine/issues/59

## Changes
- Improved RTX HDR live tuning and added a desktop SDR brightness fallback for streams that use desktop HDR conversion.
- Added recovery and troubleshooting support for the Vulkan HDR layer used by Vibeshine virtual displays.

## Fixes
- Fixed application edit and delete flows so saved apps are targeted by UUID and custom app changes are preserved.
- Prevented blocking service startup work from delaying streaming session startup.
- Preserved touch coordinates and extended virtual display topology during virtual display sessions.
- Kept the Vulkan HDR layer available for HDR streams while avoiding HDR injection when no active HDR stream is running.
- Honored custom Lossless Scaling launch paths and prevented app editing from enabling frame generation fixes unexpectedly.
- Kept RTX HDR desktop brightness and stream metadata stable when switching between desktop and app streams.
- Kept Firefox WebRTC offers from losing fallback codecs.
- Refreshed the virtual display driver package and release signing workflow for the beta.7 installer.
