# Vibepollo 1.15.5 - 2026-05-16

Notice: Vibepollo may trigger false-positive antivirus alerts. Details and workarounds are documented here: https://github.com/Nonary/vibepollo/issues/29

## Changes
- Added a release notes panel to the dashboard so the Web UI can show bundled Vibepollo notes for the installed build, nearby releases, and optional GitHub release data.
- Polished the Applications page with a clearer dashboard-style header, tighter application rows, theme-aware badges, and an improved empty state.
- Reworked Client Management into responsive cards with search and status filters, manual refresh controls, clearer pairing feedback, and matching trusted-device styling.
- Improved Windows installer version metadata so application, helper, and installer files can be compared and replaced more reliably across release revisions.
- Improved Windows Graphics Capture frame delivery to keep streaming smoother, especially in games that do not have perfect frame pacing.

## Fixes
- Fixed installer recovery for systems where registry cleaning or uninstall-cleanup tools damaged the Windows Installer cache.
- Fixed upgrades and downgrades across same-version release revisions so stale Vibepollo application and helper files are replaced reliably.
- Fixed SudoVDA installation checks so driver probing does not stall the installer on systems where the probe is slow or unresponsive.
- Fixed virtual display watchdog recovery so helper recovery work is dispatched away from the ping thread.
- Fixed persistent state and configuration writes to reduce the risk of incomplete saves during interrupted writes.
- Fixed update checks so stable respin builds respect the prerelease update toggle.
