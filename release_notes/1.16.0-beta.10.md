# Vibepollo 1.16.0-beta.10 - 2026-06-04

## Changes
- Updated the experimental Vibepollo Display Driver package to libvirtualdisplay 1.3.0.

## Fixes
- Fixed virtual display DPI retention and restoration so temporary display scaling survives reconnects more consistently.
- Fixed stale or mismatched virtual displays being reused after client changes or helper reconnects.
- Reduced display-helper restart churn during repeated helper crash loops.
- Fixed pairing and launch handling for clients that report placeholder names such as `self`.
- Fixed pairing requests with missing certificate subject names so they no longer crash.
- Fixed virtual display driver recovery when swapchains are closed after hard device loss.
