# Vibepollo 1.18.0-beta.1 - 2026-06-25

Notice: Vibepollo may trigger false-positive antivirus alerts.

## Changes
- Smoothed Windows Graphics Capture pacing under Reflex-style low-latency frame caps, keeping streams on a steadier cadence without dropping fresh frames.
- Raised virtual display refresh headroom to 4x client FPS for game-provided frame generation with capture fix, while keeping other frame-generation providers at 2x.
- Added frame pacing diagnostics for capture timing, packet queue latency, encode duration, and network rate-control behavior.

## Fixes
- Preserved WGC pacing through one short missed compositor frame instead of re-anchoring the capture cadence.
- Carried the frame-generation refresh multiplier through virtual display creation and recovery so recreated displays keep the same 4x target.
- Restored NVIDIA Control Panel frame-limiter overrides more completely after streams and logged the applied settings for easier troubleshooting.
- Prevented deferred stream-start side effects from running after a session ends during the deferral window.
