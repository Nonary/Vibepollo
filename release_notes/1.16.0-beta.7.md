# Vibepollo 1.16.0-beta.7 - 2026-05-31

Notice: Vibepollo may trigger false-positive antivirus alerts. Details and workarounds are documented here: https://github.com/Nonary/Vibepollo/issues/59

## Changes
- Updated the experimental Vibepollo Display Driver package to libvirtualdisplay 1.2.5, improving swapchain recovery during alt-tab and display presentation changes.

## Fixes
- Fixed virtual displays getting stuck on a black screen when Windows reported presentation or device loss while completing an IddCx swapchain frame.
- Fixed capture display teardown racing encoder and GPU resource cleanup, reducing crash risk during stream stop or capture reinitialization.
