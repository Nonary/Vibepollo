# Vibeshine 1.17.0-beta.2 - 2026-06-13

Notice: Vibeshine may trigger false-positive antivirus alerts. Details and workarounds are documented here: https://github.com/Nonary/vibeshine/issues/59

## Changes
- Improved RTX HDR streaming by moving NVIDIA profile polling off the frame path and applying NVIDIA profile values by default.

## Fixes
- Fixed a stream stall that could occur when RTX HDR conversion overlapped with alt-tab or foreground window swaps.
