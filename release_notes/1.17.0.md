# Vibepollo 1.17.0 - 2026-06-24

Notice: Vibepollo may trigger false-positive antivirus alerts.

## Changes
- Added opt-in NVIDIA RTX HDR streaming support for compatible NVIDIA hosts, with per-app controls and live tuning from the Web UI.
- Improved the experimental Vibepollo Virtual Display Driver with newer libvirtualdisplay updates, better DPI and display-state retention, and stronger reliability during streaming.
- Added Vulkan HDR support for virtual screens, including both SudoVDA and the experimental Vibepollo Virtual Display Driver.
- More specific changes and bug fixes are available in the changelog built into the Web UI dashboard.
