# Vibepollo 1.17.0-beta.5 - 2026-06-14

## Changes
- Added slider controls for RTX HDR brightness, contrast, saturation, and peak brightness so global and per-app tuning more closely matches NVIDIA controls.

## Fixes
- Fixed NVIDIA App RTX HDR profile detection so newer app enable bits and tuning values can be inherited when no Vibepollo override is set.
- Required explicit per-app RTX HDR opt-in before conversion starts and kept native HDR or app-enabled HDR sources from being converted again.
