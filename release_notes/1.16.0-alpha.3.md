# Vibepollo 1.16.0-alpha.3 - 2026-05-17

Notice: Vibepollo may trigger false-positive antivirus alerts. Details and workarounds are documented here: https://github.com/Nonary/Vibepollo/issues/59

## Fixes
- Fixed upgrade recovery for installs affected by alpha credential/config ACL issues, so existing Web UI credentials and paired-client state are no longer treated as missing or invalid.
- Fixed installer PATH updates to use a PowerShell registry helper, avoiding failures when the existing system PATH contains malformed quoting.
