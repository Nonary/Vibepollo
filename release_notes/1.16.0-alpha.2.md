# Vibepollo 1.16.0-alpha.2 - 2026-05-17

Notice: Vibepollo may trigger false-positive antivirus alerts. Details and workarounds are documented here: https://github.com/Nonary/Vibepollo/issues/59

## Changes
- Added full-session chart history for live streams, with quick filters for recent 3, 5, 15, 30, and 60 minute windows.

## Fixes
- Fixed an alpha.1 issue where session history database ACL hardening could lock the shared config folder, causing existing Web UI credentials to be rejected and paired clients to appear unpaired.
