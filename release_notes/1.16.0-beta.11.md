# Vibepollo 1.16.0-beta.11 - 2026-06-05

## Fixes
- Fixed WGC constant-mode capture so static content continues to repeat cached frames with fresh timing.
- Restored lower-latency WGC buffering that adapts to helper load instead of always keeping an extra frame queued.
- Fixed virtual display recovery so retained per-client displays are matched by stable identity and cannot be displaced by the encoder probe's temporary display.
- Fixed retained virtual displays on RTSP resume so preserved displays receive the helper refresh needed before capture resumes.
- Fixed anonymous or stale-client `/serverinfo` requests so discovery polling no longer crashes while privileged fields stay hidden.
- Fixed tray Restart and Quit actions so shutdown no longer aborts when the tray thread initiates exit.
- Improved Playnite app sync artwork handling so Applications refreshes avoid expensive image work and extract executable icons at native size.
- Kept NVENC teardown ordered during process shutdown so installer-triggered exits do not leave detached encoder resources running past the normal stop path.
