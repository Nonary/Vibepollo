# Vibepollo 1.16.0-beta.3 - 2026-05-28

Notice: Vibepollo may trigger false-positive antivirus alerts. Details and workarounds are documented here: https://github.com/Nonary/Vibepollo/issues/59

## Changes
- Moved the Vibepollo Display Driver selection into Display Setup Step 1 and compacted the bootstrapper when the driver option is hidden.

## Fixes
- Kept Playnite fullscreen streams alive when Playnite quits to launch a game, then restored Playnite fullscreen after the game exits instead of dropping to a dead desktop.
- Preserved existing virtual display driver choices during bootstrapper upgrades and installer migrations.
- Fixed virtual display capture crashes and shutdown deadlocks during HDR, resume, or capture reinitialization by draining D3D capture resources before teardown.
- Reset the virtual display transport when the lease watchdog cannot report failure through a parent process callback.
- Fixed RTSP binding when an IPv4 bind address is used while both address families are enabled.
- Avoided false Apollo conflict detection for unrelated applications whose names start with Apollo.
