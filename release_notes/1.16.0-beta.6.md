# Vibepollo 1.16.0-beta.6 - 2026-05-31

Notice: Vibepollo may trigger false-positive antivirus alerts. Details and workarounds are documented here: https://github.com/Nonary/Vibepollo/issues/59

## Changes
- Updated the experimental Vibepollo Display Driver package to libvirtualdisplay 1.2.4, tightening temporary-display startup, teardown, lease handling, identity rollback, and HDR profile retention.
- Added Playnite game icons to the Applications list while keeping existing box art available for clients.

## Fixes
- Fixed same-version upgrades sometimes packaging stale service and helper binaries into the MSI.
- Fixed temporary virtual display creation failures caused by invalid lease IDs.
- Delayed stale display snapshot warnings until restore failures persist across multiple attempts, reducing false warnings while users are away from the PC.
- Fixed HDR WGC capture initialization choosing 8-bit capture for HDR or Advanced Color outputs, which could leave streams oversaturated until capture restarted.
- Fixed WebRTC resume from an active RTSP stream starting a competing capture path that could leave resumed clients connected without frames.
- Fixed WGC helper startup and shutdown races that could block capture startup or teardown.
- Fixed shared texture metadata synchronization in WGC capture so frame metadata stays aligned with the copied texture under load.
