# Vibepollo 1.16.0-beta.9 - 2026-06-03

Notice: Vibepollo may trigger false-positive antivirus alerts. Details and workarounds are documented here: https://github.com/Nonary/Vibepollo/issues/59

## Changes
- Updated the experimental Vibepollo Display Driver package to libvirtualdisplay 1.2.7.

## Fixes
- Fixed virtual-display streams that use resolution remapping so the virtual display is created at the remapped size instead of staying at the client-requested transport size.
- Fixed pause-retained streams triggering display restore after Sunshine disconnects from the display helper.
- Improved virtual-display startup so streams wait for the requested display to become active before capture starts.
- Fixed stale virtual-display lease recovery and temporary-display reuse so reconnecting streams do not attach to expired or mismatched displays.
- Fixed capture stalls around UAC and secure desktop transitions by falling back to DXGI capture when WGC cannot advance.
