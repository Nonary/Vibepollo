# Vibepollo 1.16.0-beta.8 - 2026-06-02

Notice: Vibepollo may trigger false-positive antivirus alerts. Details and workarounds are documented here: https://github.com/Nonary/Vibepollo/issues/59

## Changes
- Updated the experimental Vibepollo Display Driver package to libvirtualdisplay 1.2.6.

## Fixes
- Fixed WGC capture stalls under heavy CPU or GPU load by preventing the capture helper from holding the shared WGC texture mutex while waiting on local D3D copy work.
- Improved WGC helper startup and timing diagnostics so future capture stalls identify whether startup, local D3D context waits, shared mutex waits, or shared texture publishing caused the delay.
