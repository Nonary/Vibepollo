# Vibepollo 1.16.0-beta.2 - 2026-05-28

Notice: Vibepollo may trigger false-positive antivirus alerts. Details and workarounds are documented here: https://github.com/Nonary/Vibepollo/issues/59

## Fixes
- Fixed the bundled Vibepollo Display Driver failing to install on a clean Windows machine, where Device Manager reported error Code 31 ("this device cannot start"). The driver now bundles its runtime, so it loads without requiring the Visual C++ redistributable to be installed.
