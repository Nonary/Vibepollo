# Vibepollo 1.15.4-stable.2 - 2026-05-04

Notice: Vibepollo may trigger false-positive antivirus alerts. Details and workarounds are documented here: https://github.com/Nonary/vibepollo/issues/29

## Fixes
- Fixed Playnite autosync leaving stale cached box art after cover updates.
- Fixed Playnite app IDs changing when box art is refreshed by using persistent app UUIDs for GameStream IDs.
- Fixed uninstall and factory reset behavior so user-added files in the install directory are preserved.
- Fixed HDR encoder capability detection for AMD users so HEVC/AV1 10-bit HDR support is not hidden during probing.
