# Vibeshine 1.16.0 - 2026-06-09

**Notice:** Vibeshine may trigger false-positive antivirus alerts. Details and workarounds are documented here: https://github.com/Nonary/vibeshine/issues/59

## Changes

* Added a dedicated realtime **Stats** page that turns active sessions into a full performance dashboard, with live bitrate, frame pacing, latency, encoder, network, and host metrics visible while streams are running.

* Added **session history and performance charts** for RTSP and WebRTC streams, making it easier to analyze stream behavior over time instead of relying only on logs or momentary status values.

* Added the experimental **Vibeshine Virtual Display Driver** as an opt-in Windows display driver. Compared to SudoVDA, it has shown better performance and stability in testing, with a Vibeshine-specific driver package, installer integration, faster temporary-display startup, smoother stream initialization, improved HDR and EDID handling, DPI restoration, retained per-client displays, more reliable display cleanup, and better recovery when Windows display state changes during streaming. The driver remains opt-in because display-driver behavior can vary across Windows hardware and configurations.

* Added an in-app **changelog viewer**, allowing the Web UI to display bundled release notes, installed-version context, release-line filtering, and optional GitHub release details.

* Added Game Icons to the Applications list.

* Improved **Windows Graphics Capture (WGC)** behavior and diagnostics around capture stalls.

* Fixed numerous bugs and stability issues. Detailed bug-fix notes are available in the in-app changelog.
