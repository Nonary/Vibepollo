# Vibeshine 1.16.0 - 2026-06-09

Notice: Vibeshine may trigger false-positive antivirus alerts. Details and workarounds are documented here: https://github.com/Nonary/vibeshine/issues/59

## Changes
- Added a dedicated realtime Stats page that turns active sessions into a full performance dashboard, with live bitrate, frame pacing, latency, encoder, network, and host metrics visible while streams are running.
- Added session history and performance charts for RTSP and WebRTC streams, making it easier to understand how a stream behaved over time instead of relying only on logs or momentary status values.
- Added the bundled experimental Vibeshine Virtual Display Driver as an opt-in Windows display driver. In testing it is more performant and significantly less crash-prone than SudoVDA, with a driver package built for Vibeshine, installer integration, faster temporary-display startup, smoother stream setup, and better recovery when Windows display state changes mid-stream.
- Improved the Vibeshine Virtual Display Driver experience with HDR and EDID retention, DPI restoration, retained per-client displays, and more reliable temporary-display cleanup across reconnects and multi-client use. It remains opt-in because display drivers can behave differently across Windows machines and hardware configurations.
- Added an in-app changelog so the Web UI can show bundled release notes, installed-version context, release-line filtering, and optional GitHub release details.
- Added client enable and disable controls through the Web UI and API so trusted devices can be blocked without removing their pairing.
- Added Playnite game icons to the Applications list.
- Added Web UI conveniences including logout, browse controls for selecting application executables, files, and directories, and clipboard feedback when copying troubleshooting logs.
- Added Windows-focused capture and encoding improvements, including NVENC intra-refresh, split-frame encoding support, improved WGC behavior, and better diagnostics around capture stalls.
- Improved Windows release packaging with stripped installer payloads and matching debug symbols for crash diagnostics.
- Improved overall streaming, capture, virtual display, multi-client, Playnite, installer, and shutdown stability. Detailed bug-fix notes are available in the changelog.
