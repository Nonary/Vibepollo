# Vibepollo 1.15.5-beta.1 - 2026-05-10

Notice: Vibepollo may trigger false-positive antivirus alerts. Details and workarounds are documented here: https://github.com/Nonary/Vibepollo/issues/59

## Changes
- Added a release notes panel to the dashboard so the Web UI can show bundled Vibepollo notes for the installed build, nearby releases, and optional GitHub release data.
- Polished the Applications page with a clearer dashboard-style header, tighter application rows, theme-aware badges, and an improved empty state.
- Reworked Client Management into responsive cards with search and status filters, manual refresh controls, clearer pairing feedback, and matching trusted-device styling.

## Fixes
- Fixed same-version installer upgrades so newer prerelease and dirty-build packages reliably replace stale Vibepollo application and helper files.
- Fixed Windows Graphics Capture cadence diagnostics so normal streams no longer flood helper logs while important capture-stall diagnostics remain visible.
- Fixed client and trusted-device metadata rows so icons, labels, and override counts stay aligned in narrow Web UI layouts.
