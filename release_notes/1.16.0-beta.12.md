# Vibepollo 1.16.0-beta.12 - 2026-06-05

## Changes
- Updated Windows release packaging so installer binaries are stripped while matching debug symbols remain available for crash diagnostics.
- Updated the bundled virtual display driver source package to the v1.3.0 metadata line.

## Fixes
- Fixed certificate-related crashes when unpaired or anonymous GameStream requests reached app launch, resume, cancel, app asset, and clipboard handlers before a client certificate was resolved.
- Fixed multi-client virtual display startup stalls when Windows enumerated a new virtual display before activating and applying its mode.
- Included the latest virtual display driver fixes for hard device-loss and swapchain teardown handling.
