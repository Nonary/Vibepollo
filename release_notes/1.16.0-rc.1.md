# Vibepollo 1.16.0-rc.1 - 2026-06-07

Notice: Vibepollo may trigger false-positive antivirus alerts. Details and workarounds are documented here: https://github.com/Nonary/Vibepollo/issues/59

## Changes
- Updated the experimental Vibepollo Display Driver package to libvirtualdisplay 1.3.1, improving recovery from transient swapchain access-loss events.

## Fixes
- Fixed a capture-reinitialization crash that could occur when multiple clients were connected.
- Fixed multi-client streaming so additional clients can share the active capture session instead of disrupting the current stream.
- Protected unreadable state files from being overwritten during startup or API updates.
