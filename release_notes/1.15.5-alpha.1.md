# Vibepollo 1.15.5-alpha.1 - 2026-05-04

Notice: Vibepollo may trigger false-positive antivirus alerts. Details and workarounds are documented here: https://github.com/Nonary/Vibepollo/issues/59

## Changes
- Improved Windows Graphics Capture frame pacing when games are using DLSS Frame Generation. Streams should feel much smoother now, with more WGC polish planned in the coming days.

## Fixes
- Fixed WGC IPC frame stability so queued frames stay consistent until encoding instead of being affected by later helper captures.
